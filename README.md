# 12h.run

Nederlandstalige website voor 12h.run: een kleinschalige endurance experience rond ritme, herstel en veerkracht.

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS 4
- shadcn/ui componentstijl
- Framer Motion

## Development

```bash
npm install
npm run dev
```

De hoofdsite staat op `/`. De tijdelijke donatiepagina staat op `/doneer`.
